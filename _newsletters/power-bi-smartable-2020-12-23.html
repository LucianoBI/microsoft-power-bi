News of the week

Anomaly detection (preview) is now supported in Power BI Mobile apps With the the latest Power BI Mobile apps release report consumers can view and understand anomalies directly from the mobile apps! This capability is available on all apps (iOS, Android, Windows), in all supported form factors (phone, tablet, iPad, and laptop/desktop, including Surface Hub).

↪ Microsoft

Quickly create reports in the Power BI service (preview) We’re excited to introduce a new way to quickly create reports in the Power BI service. Rather than downloading Power BI Desktop and importing the data, you’ll now be able to paste data straight into Power BI on the web and have visuals automatically generated for you.
