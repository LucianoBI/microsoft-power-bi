DAX functions of the week

CROSSJOIN (<table>, <table>[, <table>]…) Returns a table that contains the Cartesian product of all rows from all tables in the arguments. The columns in the new table are all the columns in all the argument tables.

↪ Microsoft

FIXED (<number>, <decimals>, <no_commas>) Rounds a number to the specified number of decimals and returns the result as text. You can specify that the result be returned with or without commas.
